# Unity Prefab Browser Tool

A custom Unity Editor tool designed to speed up level design by providing quick access to categorized prefabs without navigating the project folder structure.

---

## Overview

This tool provides a simple interface to organize and access prefabs by category directly inside the Unity Editor. It reduces the need to manually browse folders when building scenes.

![Visual Prefab Browser: Categorized asset management and drag-and-drop workflow](Videos/PrefabBrowser_video.gif)

---

## Solution

A custom editor window that allows developers to:

* Organize prefabs into categories
* Quickly preview them
* Drag and drop them directly into the scene

---

## Features

* Category-based organization using enums
* Drag & drop prefabs into custom categories
* Visual preview of prefabs (with fallback icon)
* Tooltip display with prefab name on hover
* Edit mode for adding and removing items
* Persistent data between sessions

---

## Usage

1. Open the tool from the Unity menu:

```
Tools/Prefab Browser
```

2. Select a category from the dropdown

3. Enable **Edit Mode** to:

   * Add prefabs via drag & drop
   * Remove existing prefabs

4. Drag prefabs directly from the window into the scene

---

## Notes

* Prefabs are organized using an enum defined in the script
* A default icon is used if no preview image is available
* If a preview isn't yet generated by Unity's cache, a default icon is provided to ensure UI stability.
